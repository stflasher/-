# Financial Planner

## Описание
Приложение для учёта личных финансов, позволяющее пользователю контролировать свои доходы и расходы, анализировать финансовые привычки и планировать бюджет.

## Установка
1. Склонируйте репозиторий.
2. Установите необходимые библиотеки: `pip install pandas matplotlib`.

## Запуск
Запустите файл `main.py` для запуска приложения.

## Функциональность
- Регистрация финансовых операций.
- Хранение данных о сумме, категории, дате и комментарии операции.
- Анализ финансовых данных с использованием pandas.
- Визуализация данных с помощью графиков.
